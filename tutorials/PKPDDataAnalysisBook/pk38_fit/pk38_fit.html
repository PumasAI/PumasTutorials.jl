Exercise PK38 - Fitting Invitro/invivo extrapolation II

2021-01-27

Objectives

In this exercise we will analyze the in vitro metabolic rate of a compound using the differential equation. We will apply nonlinear regression to five different starting concentrations. The compound is excreted completely via the hepatic route. We will model two different cytochrome P450 isoenzymes for the elimination of the compound.

The basic workflow for the estimation process is:

  1. Description of the data

  2. Exploratory analysis of the data

  3. Pharmacokinetic modelling

  4. Diagnostic Plots

  5. Validation

  6. Simulation

  • Concentration = uM

  • Dose (amt) = μmol/L

  • Group (1 : Dose 50, 2 : Dose 30, 3 : Dose 10, 4 : Dose 3, 5 : Dose 1)

Double-Enzyme NonLinear Elimination Model

The volume of the incubation medium is set to 1ml.

pk_38        = @model begin
  @param begin
    tvvmax1  ∈ RealDomain(lower=0)
    tvkm1    ∈ RealDomain(lower=0)
    tvvmax2  ∈ RealDomain(lower=0)
    tvkm2    ∈ RealDomain(lower=0)
    Ω        ∈ PDiagDomain(2)
    σ_add    ∈ RealDomain(lower=0)
  end

  @random begin
    η        ~ MvNormal(Ω)
  end

  @pre begin
    Vmax1    = tvvmax1 * exp(η[1])
    Km1      = tvkm1
    Vmax2    = tvvmax2 * exp(η[2])
    Km2      = tvkm2
    Vmedium  = 1
  end

  @vars begin
    VMKM    := ((Vmax1*Central/(Km1+Central))+(Vmax2*Central/(Km2+Central)))
  end

  @dynamics begin
    Central' =  -VMKM/Vmedium
  end

  @derived begin
    cp       = @. Central/Vmedium
    dv       ~ @. Normal(cp, σ_add)
  end
end

function gof(pktvp_mr_inspect_run2)
  p1 = plot()
  @df pktvp_mr_inspect_run2 scatter!(
      p1,
      :dv_pred, :dv;
      ylabel = "Observed dv (uM)",
      xlabel = "Population Predicted (uM)",
      label = "",
      legend=false,
